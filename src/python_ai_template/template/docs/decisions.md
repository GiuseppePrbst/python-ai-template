# Decisiones arquitectónicas (ADR)

> **Nota**: las decisiones del repositorio generador (`python-ai-template`)
> se mantienen en el repositorio de la plantilla y no se duplican aquí.
> Este archivo registra únicamente decisiones arquitectónicas **propias de
> este proyecto generado**. Si una decisión ya está documentada en el
> repositorio de la plantilla, se referencia aquí por su ADR.

Registro de decisiones arquitectónicas del proyecto. Cada entrada sigue el formato ADR ligero: ID, fecha, contexto, decisión, alternativas, consecuencias, reversibilidad, estado.

Las decisiones no se modifican silenciosamente. Una corrección se hace añadiendo una nueva entrada que referencia y corrige la anterior.

---

## Plantilla

```markdown
## ADR-XXX: <título corto>

- **Fecha**: YYYY-MM-DD
- **Estado**: propuesta | aceptada | rechazada | superseded by ADR-XXX.
- **Contexto**: qué situación o problema motiva la decisión.
- **Decisión**: qué se ha decidido.
- **Alternativas consideradas**:
  - Alternativa A: motivo de descarte.
  - Alternativa B: motivo de descarte.
- **Consecuencias**: efectos esperados, tanto positivos como negativos.
- **Reversibilidad**: fácil | difícil | irreversible.
```
